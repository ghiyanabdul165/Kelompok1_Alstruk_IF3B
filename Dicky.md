Pada program ini terdapat beberapa contoh tipe data yang umum digunakan di Python. Pertama ada tipe data integer, yaitu angka bulat tanpa koma, seperti pada contoh jumlah_barang = 12. Tipe data ini dipakai untuk nilai yang pasti dan tidak membutuhkan desimal.

Selanjutnya adalah tipe data float, yaitu angka yang memiliki koma. Contohnya berat_badan = 58.6. Float digunakan ketika data membutuhkan angka pecahan, misalnya berat, tinggi, atau nilai perhitungan tertentu.

Program juga menggunakan tipe data string, yaitu data berupa teks. Contohnya nama_siswa = "Andi". String cocok dipakai untuk menyimpan tulisan seperti nama atau kalimat.

Tipe data berikutnya adalah boolean, yaitu tipe data yang hanya berisi nilai True atau False. Misalnya is_online = True dan is_admin = False. Boolean biasanya digunakan untuk menunjukkan kondisi, seperti apakah pengguna sedang aktif atau tidak.

Selain itu, terdapat tipe data kompleks seperti 8 + 2j yang terdiri dari angka real dan imajiner. Tipe data ini biasanya digunakan dalam perhitungan matematika tertentu.

Terakhir, program menampilkan tipe data dari bahasa C menggunakan module ctypes, seperti c_long dan c_float. Tipe data ini dipakai ketika Python perlu berhubungan dengan sistem atau library yang menggunakan bahasa C.
