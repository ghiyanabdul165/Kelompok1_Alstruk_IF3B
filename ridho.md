1. input()
Ini fungsinya untuk nanyain sesuatu ke user.
Jadi program berhenti dulu sampai user ngisi jawaban.

2. nama = input("Masukkan nama kamu: ")
Program bakal nanya, dan apa pun yang user ketik, masuk ke variabel nama.

3. usia = int(input(...))
Karena input() hasilnya selalu teks,
maka kita ubah dulu jadi angka bulat pakai int().

4. tinggi = float(input(...))
Kalau butuh angka desimal, pakai float().

5. is_student = is_student.lower() == "ya"
Ini trik supaya jawaban user seperti “Ya”, “YA”, “ya” semuanya dianggap sama → jadi True.
